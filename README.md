# Интерпретируемая диагностика пневмонии по рентгеновским снимкам грудной клетки с использованием зонально-зависимого анализа внимания

## English

This repository contains the code and materials for the research project on interpretable pneumonia detection from chest X-ray images using deep learning with zone-aware attention analysis.

### Overview

Pneumonia is a leading cause of morbidity worldwide. Automated diagnosis using deep learning shows promise, but clinical adoption requires model interpretability. This project investigates ResNet-based architectures enhanced with Grad-CAM and zone-aware attention mechanisms to provide clinically meaningful visual explanations.

### Repository Structure

| Folder | Description |
|--------|-------------|
| `data/` | Dataset information and preprocessing scripts |
| `paper/` | Research papers and literature review |
| `writing/` | Manuscript drafts and supplementary materials |

### Methods

- **ResNet** backbone for feature extraction
- **Grad-CAM** for visual explanation of model decisions
- **Zone-aware attention** analysis focusing on clinically relevant chest regions

### Dataset

Chest X-ray images from public datasets (e.g., NIH ChestX-ray14, RSNA Pneumonia Detection).

## Русский

Данный репозиторий содержит код и материалы исследовательского проекта по интерпретируемому обнаружению пневмонии на рентгеновских снимках грудной клетки с использованием глубокого обучения и зонально-зависимого анализа внимания.

### Обзор

Пневмония является одной из ведущих причин заболеваемости во всём мире. Автоматизированная диагностика с использованием глубокого обучения показывает перспективные результаты, однако клиническое внедрение требует интерпретируемости моделей. В данном проекте исследуются архитектуры на основе ResNet, дополненные методами Grad-CAM и механизмами зонально-зависимого внимания, для обеспечения клинически значимых визуальных объяснений.

## 中文

本仓库包含基于深度学习和区域感知注意力分析的可解释胸部X光肺炎检测研究项目的代码与材料。

### 项目概述

肺炎是全球范围内导致发病的主要原因之一。基于深度学习的自动化诊断展现出良好前景，但临床落地需要模型具备可解释性。本项目研究基于 ResNet 的架构，结合 Grad-CAM 和区域感知注意力机制，以提供具有临床意义的视觉解释。

### 仓库结构

| 文件夹 | 说明 |
|--------|------|
| `data/` | 数据集信息与预处理脚本 |
| `paper/` | 研究论文与文献综述 |
| `writing/` | 手稿草稿与补充材料 |

### 方法

- **ResNet** 骨干网络用于特征提取
- **Grad-CAM** 用于模型决策的视觉解释
- **区域感知注意力**分析，聚焦于临床相关的胸部区域

### 数据集

来自公开数据集的胸部X光图像（如 NIH ChestX-ray14、RSNA Pneumonia Detection）。

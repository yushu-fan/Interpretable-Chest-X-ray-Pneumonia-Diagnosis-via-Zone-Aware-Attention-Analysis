# 课程论文实施计划（Курсовая работа）

## 目标
完成并提交完整课程论文（курсовая работа）+ 答辩PPT，含真实实验结果。

## 交付物
| 文件 | 说明 |
|------|------|
| `курсовая_работа.docx` | 完整课程论文（俄语，含中文注释）|
| `презентация_защиты.pptx` | 答辩PPT（俄语正文 + 中文备注）|
| `results/` | 实验结果（模型权重、Grad-CAM 热力图、指标图表）|

## 论文结构（курсовая_работа.docx）
1. **Введение**（引言）— 研究背景、目标、任务
2. **Обзор литературы**（文献综述）— 基于 literature_review.md，精选 8-10 篇核心文献
3. **Материалы и методы**（材料与方法）— ChestXRay2017 数据集描述、预处理流程、ResNet-18 架构、训练配置
4. **Эксперименты и результаты**（实验与结果）— **含真实训练结果**、混淆矩阵、ROC 曲线、Grad-CAM 可视化
5. **Обсуждение**（讨论）— 结果分析、与文献对比、局限性
6. **Заключение**（结论）— 主要成果、未来工作
7. **Список литературы**（参考文献）— ГОСТ Р 7.0.5–2008 格式

## 实施步骤

### Step 1: 数据准备
- 检查 ChestXRay2017 是否已下载（Kaggle: `chest-xray-pneumonia`）
- 若无数据，编写下载脚本（kaggle API 或手动下载指引）
- 数据集统计：5,856 张，Normal / Pneumonia 二分类，划分 train/val/test

### Step 2: 训练脚本（train.py）
- 基于 ResNet-18（ImageNet 预训练）
- 数据增强：Resize(224), RandomHorizontalFlip, Normalize(ImageNet stats)
- 优化器：Adam(lr=1e-3)，CrossEntropyLoss
- 训练 20-30 epochs，保存最佳模型权重
- 输出：accuracy, AUC, precision, recall, F1 每 epoch 日志

### Step 3: Grad-CAM 可视化（gradcam.py）
- 实现 Grad-CAM 热力图生成
- 对测试集随机样本生成叠加图（原图 + 热力图）
- 保存为 PNG，用于论文插图

### Step 4: 评估与绘图（evaluate.py）
- 在测试集上计算最终指标（Accuracy, AUC, Precision, Recall, F1）
- 生成：混淆矩阵、ROC 曲线、训练 loss/acc 曲线
- 保存所有图表为 PNG（用于论文）

### Step 5: 撰写课程论文（write_coursework.py）
- 生成 `курсовая_работа.docx`（俄语 + 中文注释）
- 第 4 章（实验结果）**动态插入**真实训练结果和图表
- 图表以 base64 嵌入或本地引用方式插入 DOCX

### Step 6: 制作答辩 PPT（make_defense_pptx.py）
- 生成 `презентация_защиты.pptx`（俄语正文 + 中文备注）
- 约 12-15 页：标题、引言、方法、实验设置、结果（含图表）、讨论、结论
- 备注页写中文，方便你理解每页要点

## 关键决策点（需用户确认）
1. **数据来源**：ChestXRay2017 是否已下载？若否，是否用 Kaggle API 自动下载？
2. **训练设备**：CPU 训练（torch 2.10.0+cpu 已安装），预计 20 epochs 约需 1-2 小时，是否接受？
3. **论文字数**：МГУ 课程论文字数要求约多少页（通常 25-40 页）？
4. **Grad-CAM 实现**：用 pytorch-grad-cam 库还是手动实现？

## 风险
- 数据集下载可能受网络限制（Kaggle 需认证）
- CPU 训练时间较长，需耐心等待
- 若训练结果不理想（accuracy < 85%），需调整超参或数据增强

## 1. Обоснование темы и научная новизна

### 1.1 Проблема

Глубокое обучение достигло высоких показателей в классификации пневмонии по рентгеновским снимкам (CheXNet, 2017; Kermany et al., 2018). Однако перед клиническим внедрением стоят две ключевые проблемы:

1. **Необъяснимость («чёрный ящик»)**: врачи не могут оценить обоснованность решения модели.
2. **Отсутствие оценки уверенности**: модель выдаёт вероятность для любого снимка, не различая надёжные и сомнительные случаи.

### 1.2 Научная новизна (три пункта)

| № | Элемент новизны | Отличие от существующих работ |
|---|---------------------|------------------------------------------|
| 1 | **Зональный Grad-CAM** — количественная оценка интерпретируемости по анатомическим зонам лёгких | Существующие работы (Zone-Aware, 2026) используют зоны для улучшения классификации; мы впервые вводим **метрики интерпретируемости на уровне зон** (RAS, кросс-зональная энтропия). |
| 2 | **Оценка неопределённости (Monte Carlo Dropout)** с обоснованным порогом принятия решения | Большинство работ по chest X-ray не оценивают неопределённость; CXR-NeXus (2026) требует сложного обучения с прототипами; наш подход — **простая постобработка без дообучения**. |
| 3 | **Комбинированная стратегия**: зональная интерпретируемость + отказ от предсказания при низкой уверенности | Впервые два направления объединяются: врач получает **и** визуализацию зон внимания, **и** сигнал о необходимости ручной проверки. |

## 2. Исследовательский вопросы

1. **Вопрос интерпретируемости:** Как копичественно оценить, насколько зоны внимания модели (Grad-CAM) соответствуют анатомически обоснованному распределению пневмонии по долям лёгких?
2. **Вопрос надёжности:** При каком пороге неопределённости модель может «отказаться» от предсказания и передать снимок врачу, не снижая общую точность?
3. **Вопрос обобщающей способности:** Улучшает ли добавление зональной интерпретируемости и оценки неопределённости переносимость модели на другой набор данных (Zero-shot / Cross-dataset)?

## 3. Методология и технический план

### 3.1 Наборы данных

| Набор данных | Описание | Назначение |
|--------------|-----------|-------------|
| **ChestXRay2017** (Kermany et al., 2018) | 5 856 снимков, Noral / Pneumonia | Основной набор для обучения и теста |
| **RSNA Pneumonia Detection** | ~30 000 DICOM-снимков | Тест на переносимость (без переобучения) |
| **Подмножество с врачебной разметкой** (100 снимков) | Вручную размеченные зоны поражения | Вычисление метрики RAS (Region Alignment Score) |

### 3.2 Базовая модель

- Архитектура: **ResNet-18**, предобученная на ImageNet
- Заморозка: первые 3 блока ResNet заморожены, 4-й блок и FC-слой дообучаются
- Функция потерь: Cross Entropy Loss → Focal Loss (для борьбы с дисбалансом классов)
- Оптимизатор: Adam, lr = 10⁻³, план снижения learning rate

### 3.3 Инновация №1: Зональный Grad-CAM

**Шаги реализации:**

1. **Сегментация зон лёгких**
   - Использование предобученной U-Net на снимках грудной клетки
   - Разделение каждого лёгкого на 3 зоны: верхняя доля / средняя доля / нижняя доля
   - Итого: 6 зон для каждого снимка

2. **Вычисление зонального внимания**
   - Генерация Grad-CAM тепловой карты для каждого снимка
   - Наложение зональных масок: получение среднего значения внимания в каждой зоне
   - Итоговый вектор: **6-мерный вектор зонального внимания** для каждого снимка

3. **Метрики интерпретируемости (новые!)**
   - **RAS (Region Alignment Score):** IoU между зонами высокого внимания модели и зонами, размеченными врачом
   - **CZAE (Cross-Zone Attention Entropy):** энтропия распределения внимания по зонам (высокая энтропия = модель «размыта» в решениях, возможен переобучение)
   - **Сравнение с Zone-Aware 2026:** у них повышение Accuracy за счёт зон; у нас — повышение интерпретируемости (RAS)

4. **Визуализация**
   - Тепловые карты Grad-CAM по зонам (отдельные карты для каждой зоны)
   - Гистограмма зонального внимания (bar chart) для каждого снимка

### 3.4 Инновация №2: Оценка неопределённости

**Шаги реализации:**

1. **Monte Carlo Dropout**
   - В FC-слоях сохраняется Dropout = 0.5 во время теста
   - Для каждого снимка выполняется **T = 50** прямых проходов (каждый раз случайное отключение нейронов)
   - Получение 50 предсказаний вероятности → вычисление среднего μ и дисперсии σ²

2. **Количественная оценка неопределённости**
   - **Энтропия:** `H = −Σ p_i log(p_i)`, высокая энтропия = высокая неопределённость
   - **Дисперсия:** `Var = (1/T)·Σ(p_t − μ)²`, высокая дисперсия = высокая неопределённость
   - **Порог неопределённости:** σ² > θ (оптимальный θ находится на валидационном наборе по метрике Conditional Accuracy)

3. **Стратегия принятия решения**
   - Если σ² ≤ θ: модель выдаёт предсказание (высокая уверенность)
   - Если σ² > θ: модель выводит «Требуется проверка врача» (неопределённость)

4. **Метрики оценки**
   - **Coverage:** доля снимков, для которых модель дала определённый ответ
   - **Conditional Accuracy:** точность только на снимках с высокой уверенностью (должна быть выше общей Accuracy)
   - **Review Rate:** доля снимков, переданных врачу
   - **AUC-ROC (на снимках с высокой уверенностью):** проверка того, что неопределённость действительно отсеивает «сложные» снимки

### 3.5 План экспериментов (6 экспериментов)

| № | Конфигурация модели | Цель |
|---|----------------------|------|
| Exp-1 | ResNet-18 Basline (без Grad-CAM, без неопределённости) | Базовая линия |
| Exp-2 | ResNet-18 + Grad-CAM (глобальный) | Проверка эффекта визуализации |
| Exp-3 | ResNet-18 + Зональный Grad-CAM (инновация №1) | **Основной эксперимент** |
| Exp-4 | ResNet-18 + Monte Carlo Dropout (инновация №2) | **Основной эксперимент** |
| Exp-5 | ResNet-18 + CBAM + Focal Loss | Сравнение с SOTA по Accuracy |
| Exp-6 | ResNet-18 + Зональный Grad-CAM + Неопределённость (комбинация) | **Полная модель** |

**Дополнительный тест:** Cross-dataset переносимость (обучение на ChestXRay2017 → тест на RSNA) для моделей Exp-3 и Exp-6.
